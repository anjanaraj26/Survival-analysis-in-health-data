# Notebooks Overview

This folder contains Jupyter notebooks used for exploratory data analysis,
feature engineering, and survival model development.

## Notebook Structure

1. `01_eda.ipynb`
   - Data understanding
   - Missing values analysis
   - Descriptive statistics
   - Initial survival curves

2. `02_modeling.ipynb`
   - Cox Proportional Hazards model
   - Assumption checks
   - Random Survival Forest

3. `03_evaluation.ipynb`
   - Model comparison
   - C-index
   - Calibration plots
